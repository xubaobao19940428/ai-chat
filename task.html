AsyncTaskRecordController API 文档

概述

管理用户的 AI 异步任务记录（app_ai_async_tasks 表），提供任务列表、详情、点赞、删除等接口。

注意：任务生成的文件（图片/视频/音频）存储在 app_u_user_files 表，通过 source_type=ai_task 关联，可通过 FileController 接口查询。

路径前缀为 /v1，所有接口均需要登录。

1. 获取任务列表

  • 接口描述：分页获取当前用户的异步任务列表
  • HTTP 方法：GET
  • 请求路径：/v1/async-task-record/list
  • 是否需要登录：是

1.1 请求参数

参数名 位置 类型 必填 默认值 说明
capability query string 否 - 能力类型筛选，如 video、image、audio_generation
status query int 否 - 状态筛选：0-处理中，1-成功，-1-失败
page query int 否 1 页码，最小 1
page_size query int 否 20 每页数量，最少 1、最多 50

1.2 响应说明

成功时返回任务列表和分页信息：

{
  "code": 0,
  "message": "success",
  "data": {
    "list": [
      {
        "id": 123,
        "provider_task_id": "task_abc123",
        "provider_id": 1,
        "character_id": 10,
        "model_id": 5,
        "capability": "video",
        "user_id": 1001,
        "status": 1,
        "progress": 100,
        "task_params_json": "{\"prompt\":\"一只猫在草地上奔跑\"}",
        "task_output_json": "[{\"type\":\"video\",\"url\":\"https://...\"}]",
        "thumbnail": "https://cdn.example.com/thumb.jpg",
        "localized": 1,
        "like_count": 5,
        "error": null,
        "started_at": 1710000000,
        "completed_at": 1710000100,
        "created_at": 1710000000,
        "updated_at": 1710000100
      }
    ],
    "page": 1,
    "page_size": 20,
    "total": 150
  }
}

字段说明：

  • id：任务记录ID
  • provider_task_id：Provider 返回的任务ID
  • capability：能力类型
  • status：任务状态（0-处理中，1-成功，-1-失败）
  • progress：进度百分比（0-100）
  • thumbnail：首张缩略图URL
  • localized：本地化状态（0-未处理，1-已完成，2-失败）
  • like_count：点赞数
  • total：符合条件的任务总数

2. 获取任务详情

  • 接口描述：获取单个任务的详细信息，包含关联的文件列表
  • HTTP 方法：GET
  • 请求路径：/v1/async-task-record/detail
  • 是否需要登录：是

2.1 请求参数

参数名 位置 类型 必填 说明
id query int 是 任务ID

2.2 响应说明

成功时返回任务详情和关联文件：

{
  "code": 0,
  "message": "success",
  "data": {
    "id": 123,
    "provider_task_id": "task_abc123",
    "capability": "video",
    "status": 1,
    "progress": 100,
    "task_params_json": "{\"prompt\":\"一只猫在草地上奔跑\"}",
    "task_output_json": "[{\"type\":\"video\",\"url\":\"https://...\"}]",
    "thumbnail": "https://cdn.example.com/thumb.jpg",
    "localized": 1,
    "like_count": 5,
    "created_at": 1710000000,
    "files": [
      {
        "id": 456,
        "file_name": "video_output.mp4",
        "file_url": "https://cdn.example.com/video.mp4",
        "file_type": "video",
        "source_index": 0,
        "created_at": 1710000100
      }
    ]
  }
}

files 字段：关联的文件列表，按 source_index 排序

2.3 错误码

错误码 message 说明
40001 param_error 参数错误（id 无效）
30001 task_not_found 任务不存在
20001 permission_denied 任务不属于当前用户

3. 点赞/取消点赞任务

  • 接口描述：对任务进行点赞或取消点赞（任务级别，非单个文件）
  • HTTP 方法：POST
  • 请求路径：/v1/async-task-record/like
  • 是否需要登录：是

3.1 请求参数

参数名 位置 类型 必填 默认值 说明
id body int 是 - 任务ID
action body string 是 like 操作类型：like 点赞，unlike 取消点赞

3.2 请求示例

{
  "id": 123,
  "action": "like"
}

3.3 响应说明

成功时返回空数据：

{
  "code": 0,
  "message": "success",
  "data": []
}

3.4 错误码

错误码 message 说明
40001 param_error 参数错误
30001 task_not_found 任务不存在
20001 permission_denied 任务不属于当前用户（无权点赞）
30001 operation_failed 操作失败

4. 删除任务记录

  • 接口描述：删除任务记录（同时删除关联的文件记录）
  • HTTP 方法：POST
  • 请求路径：/v1/async-task-record/delete
  • 是否需要登录：是

4.1 请求参数

参数名 位置 类型 必填 说明
id body int 是 任务ID

4.2 请求示例

{
  "id": 123
}

4.3 响应说明

成功时返回：

{
  "code": 0,
  "message": "success",
  "data": []
}

4.4 错误码

错误码 message 说明
40001 param_error 参数错误
30001 task_not_found 任务不存在
20001 permission_denied 任务不属于当前用户
30001 operation_failed 删除失败

附录：任务状态说明

任务状态（status）

值 说明
0 处理中
1 成功
-1 失败

本地化状态（localized）

值 说明
0 未处理
1 已完成
2 失败

能力类型（capability）

常见值：video、image、audio_generation、image_edit、video_edit 等

相关接口

  • 查询任务生成的文件：使用 GET /v1/files?source_type=ai_task&source_id={任务ID}
  • 删除任务生成的文件：使用 POST /v1/files/delete-by-source
