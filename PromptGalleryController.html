PromptGalleryController API 文档

概述

提供提示词画廊的浏览、筛选和收藏相关接口。路径前缀为 /v1。

  • 列表、详情、筛选选项接口可在未登录状态使用
  • 收藏列表接口需要登录
  • 支持多语言，通过请求头 Accept-Language 指定语言代码

1. 获取画廊列表

  • 接口描述：分页获取提示词画廊列表，支持多种筛选条件
  • HTTP 方法：GET
  • 请求路径：/v1/prompt-gallery
  • 是否需要登录：否

1.1 请求参数（Query）

参数名 类型 必填 默认值 说明
page int 否 1 页码，最小 1
page_size int 否 20 每页数量，默认 20，最大 100
model string 否 - 模型名称筛选，精确匹配
type string 否 - 类型筛选：image/video/audio
is_featured int 否 - 编辑精选筛选：0/1
tag string 否 - 标签筛选，模糊匹配

1.2 响应说明

成功时返回列表和分页信息：

{
  "code": 0,
  "message": "success",
  "data": {
    "list": [
      {
        "id": 123,
        "type": "image",
        "model": "flux-pro",
        "original_url": "https://...",
        "author": "作者名",
        "thumbnail": "https://...",
        "description": "描述文本",
        "prompt": "提示词内容",
        "config": {
          "width": 1024,
          "height": 1024,
          "steps": 20
        },
        "assets": [
          {
            "url": "https://...",
            "type": "image"
          }
        ],
        "tags": ["风景", "写实"],
        "is_featured": 1,
        "created_at": 1234567890
      }
    ],
    "page": 1,
    "page_size": 20
  }
}

字段说明：

  • config：生成配置参数（JSON 对象）
  • assets：相关资源文件列表
  • tags：标签数组
  • description 和 prompt：根据 Accept-Language 返回对应语言版本，优先级：指定语言 > 英语 > 中文

1.3 缓存说明

  • 列表数据缓存 1 小时
  • 缓存键包含所有筛选条件和语言参数

2. 获取画廊详情

  • 接口描述：获取指定提示词画廊的详细信息
  • HTTP 方法：GET
  • 请求路径：/v1/prompt-gallery/{id}
  • 是否需要登录：否

2.1 请求参数

参数名 位置 类型 必填 说明
id path int 是 画廊 ID

2.2 响应与错误

  • ID 无效：invalid_gallery_id（错误码 40001）
  • 画廊不存在：gallery_not_found（错误码 30001）

成功时返回完整的画廊数据，包含 updated_at 字段：

{
  "code": 0,
  "message": "success",
  "data": {
    "id": 123,
    "type": "image",
    "model": "flux-pro",
    "original_url": "https://...",
    "author": "作者名",
    "thumbnail": "https://...",
    "description": "描述文本",
    "prompt": "提示词内容",
    "config": {},
    "assets": [],
    "tags": [],
    "is_featured": 1,
    "created_at": 1234567890,
    "updated_at": 1234567890
  }
}

2.3 缓存说明

  • 详情数据缓存 2 小时
  • 缓存键包含 ID 和语言参数

3. 获取筛选选项

  • 接口描述：获取可用的筛选选项（模型列表和类型列表）
  • HTTP 方法：GET
  • 请求路径：/v1/prompt-gallery/filters
  • 是否需要登录：否

3.1 请求参数

无

3.2 响应说明

成功时：

{
  "code": 0,
  "message": "success",
  "data": {
    "models": ["flux-pro", "flux-dev", "stable-diffusion-3"],
    "types": ["image", "video"]
  }
}

说明：

  • models：当前画廊中所有可用的模型名称列表
  • types：当前画廊中所有可用的类型列表

3.3 缓存说明

  • 筛选选项缓存 1 小时

4. 获取收藏的提示词列表

  • 接口描述：获取当前用户收藏的提示词画廊列表
  • HTTP 方法：GET
  • 请求路径：/v1/prompt-gallery/favorites
  • 是否需要登录：是

4.1 请求参数（Query）

参数名 类型 必填 默认值 说明
type string 是 - 收藏类型：prompt_image/prompt_video
page int 否 1 页码，最小 1
page_size int 否 20 每页数量，默认 20，最大 100

4.2 响应与错误

  • 收藏类型无效：invalid_favorite_type（错误码 40001）

成功时：

{
  "code": 0,
  "message": "success",
  "data": {
    "list": [
      {
        "id": 123,
        "type": "image",
        "model": "flux-pro",
        "original_url": "https://...",
        "author": "作者名",
        "thumbnail": "https://...",
        "description": "描述文本",
        "prompt": "提示词内容",
        "config": {},
        "assets": [],
        "tags": [],
        "is_featured": 1,
        "created_at": 1234567890,
        "favorited_at": 1234567900
      }
    ],
    "page": 1,
    "page_size": 20
  }
}

字段说明：

  • 返回的数据结构与画廊列表相同
  • 额外包含 favorited_at 字段，表示收藏时间
  • 按收藏时间倒序排列
  • 如果收藏的提示词已被删除，则不会出现在列表中

4.3 实现逻辑

  1. 从收藏表获取用户收藏的 target_id 列表（分页）
  2. 批量查询提示词画廊数据
  3. 将收藏时间附加到每个提示词数据中
  4. 返回完整的提示词信息

多语言支持

所有接口支持通过 HTTP 请求头指定语言：

Accept-Language: zh-CN

支持的语言代码：

  • zh-CN：简体中文
  • en：英语
  • 其他语言代码

翻译优先级：

  1. 请求的语言
  2. 英语（en）
  3. 简体中文（zh-CN）

如果指定语言的翻译不存在，会自动降级到下一优先级语言。

使用示例

获取图片类型的提示词列表

curl -X GET "https://api.example.com/v1/prompt-gallery?type=image&page=1&page_size=20" \
  -H "Accept-Language: zh-CN"

获取编辑精选的提示词

curl -X GET "https://api.example.com/v1/prompt-gallery?is_featured=1" \
  -H "Accept-Language: zh-CN"

获取我收藏的图片提示词

curl -X GET "https://api.example.com/v1/prompt-gallery/favorites?type=prompt_image&page=1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Accept-Language: zh-CN"

按模型筛选

curl -X GET "https://api.example.com/v1/prompt-gallery?model=flux-pro" \
  -H "Accept-Language: zh-CN"

按标签筛选

curl -X GET "https://api.example.com/v1/prompt-gallery?tag=风景" \
  -H "Accept-Language: zh-CN"
